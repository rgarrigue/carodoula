# carodoula

Un site web pour une Doula, réalisé via [Hugo](https://gohugo.io/), avec le thème [Hugo Story](https://themes.gohugo.io/theme/hugo-story/).

## Publication

Mis en ligne via https://pages.cloudflare.com/. Suivre le tutorial, puis ajouter une variable d'environnement HUGO_VERSION=0.81.1
